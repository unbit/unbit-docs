Cos'e' ?

Unbit Ã¨ una piattaforma per il deploy di applicazioni web dedicata agli sviluppatori e alle aziende del settore.
Una volta attivato un Container (un'area dedicata della memoria di un singolo server) potrai far girare diversi tipi di applicazioni all'interno dello stesso.

I server sono forniti con preistallati centinaia di pacchetti in molteplici versioni, ma puoi anche compilare eventuali prodotti non disponibili.
Ogni aggiornamento dei pacchetti di sistema e' incrementale, quindi potrai aggiornare alle nuove versioni solo quando ne avrai la necessita' (o la voglia).

Puoi acquistare piu' Container in un singolo account e, cosa molto piu' importante, puoi attivare Container su server diversi per implementare soluzioni ad alta scalabilita' ed affidabilita'. Vai alla documentazione


uWSGI

Il progetto uWSGI e' il cuore di Unbit.
E' il server di default per le applicazioni WSGI, Rack, PSGI, PHP, CGI, Lua-WSAPI e Go.
Fornisce inoltre i proxy e i load-balancer per tutta l'infrastruttra, nonche' le funzionalita' di monitoraggio ormai necessarie in ogni applicazione di rete.
Prodotto di punta di Unbit, offre un ambiente solido ed estremamente configurabile anche in soluzioni distribuite.
uWSGI e' un prodotto OpenSource (licenza GPL2), quindi potrai facilmente ricostruire l'ambiente Unbit sui tuoi sistemi di sviluppo.
