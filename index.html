Cos'e' ?

unbit.it Ã¨ una piattaforma per il deploy di applicazioni web dedicata agli sviluppatori e alle aziende del settore.
E' basata sulla tecnologia uwsgi.it che combina i container Linux con l'application server multilinguaggio uWSGI. Un container e' costituito da un quantitativo di memoria ram, una quota disco e da uno strato applicativo che semplifica le procedure di gestione dei servizi e il loro monitoraggio, inoltre offre un livello versatilita' e sicurezza maggiore rispetto ai classici sistemi di hosting condiviso.


uWSGI

Il progetto uWSGI e' il cuore di unbit.

E' il server di default per le applicazioni WSGI, Rack, PSGI, PHP, CGI, Lua-WSAPI e Go.
Fornisce inoltre i proxy e i load-balancer per tutta l'infrastruttra, nonche' le funzionalita' di monitoraggio ormai necessarie in ogni applicazione di rete.

Prodotto di punta di Unbit, offre un ambiente solido ed estremamente configurabile anche in soluzioni distribuite.
uWSGI e' un prodotto OpenSource (licenza GPL2).
Se sei interessato al supporto commerciale (o l'acquisto di licenze) consulta il sito unbit.com


uwsgi.it

Il progetto uwsgi.it e' la nostra piattaforma di hosting realizzata specificatamente per le applicazioni di nuova generazione basate su Python, Perl, Ruby, Lua, Go, Node, Java...

E' basata su uWSGI (ovviamente) e completamente opensource. La configurazione avviene tramite una semplice api HTTP/REST (per la quale sono disponibili un fronted HTML5 su https://uwsgi.it e wrapper per vari linguaggi)

E' indipendente dal datacenter, quindi potrai creare sistemi ad alta affidabilita' o disaster recovery con poche righe di configurazione.
Quando attivi un container uwsgi.it puoi scegliere in quale datacenter attivarlo tra quelli disponibili elencati sul listino (ti basta comunicarlo allo staff).

Se disponi gia' di hardware o di server dedicati (anche su cloud), puoi delegarne la gestione al nostro staff che provvedera' a configurare i tuoi server come nodi della nostra infrastruttura uwsgi.it.

Un semplice quickstart e' disponibile a questa url: https://github.com/unbit/uwsgi.it/blob/master/CustomerQuickstart.md

Tecnologie supportate

La nostre piattaforme di hosting supportano moltissime tecnologie (anche non mainstream).
Questo e' un elenco (incompleto) di cio' che puoi eseguire in un container uwsgi.it:

Django, RubyOnRails, Flask, Web2Py, Sinatra, Zope, PostgreSQL, Tomcat, Jetty, node.js, redis, memcached
beanstalkd, Apache http, CouchDB, php via mod_php o uWSGI, git, mercurial, MongoDB, MySQL, ElasticSearch
Catalyst, Mojolicious, bottle, web.py

Inoltre il supporto per HTTPS/SNI, websockets e streaming Icecast2 e' attivo per tutte le istanze.
